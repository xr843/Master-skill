# 法师信息录入

你正在帮助用户创建一位有可靠史料与合规原典来源的佛教祖师大德 AI 教学角色，适用于印度、汉传、藏传与南传传统。请逐一提问，收集以下信息。

## 问题一：法师名称（必填）

请问您想创建哪位祖师大德的教学角色？

> 请提供法师的全名（如"慧能大师"、"阿底峡尊者"、"觉音尊者"）。
> 我会自动从 FoJin 知识图谱中匹配该人物的宗派、时代、师承等信息。

**处理规则：**
- 收到名称后，调用 `fojin_bridge.search_kg_entities(name, entity_type="person")` 匹配
- 若匹配成功：展示匹配到的信息（名称、宗派、时代），请用户确认
- 若匹配失败：请用户手动提供传统与宗派（印度 / 汉传 / 藏传 / 南传及其具体传承）、时代、师承

## 问题二：关注方面

您最关注这位法师的哪些方面？

- [ ] 核心教义与思想体系
- [ ] 修行方法与实践指导
- [ ] 特定经典的讲解风格
- [ ] 全部（默认）

**处理规则：**
- 默认选择"全部"
- 若用户选择特定经典，追问具体经名

## 问题三：语言偏好

生成的教学角色使用什么语言？

- [ ] 中文（默认）
- [ ] 中文 + 梵文术语保留（适合印度、唯识、律宗）
- [ ] 中文 + 保留文言引文（适合禅宗、天台、华严）
- [ ] 中文 + 藏文 / Wylie 术语保留（适合藏传）
- [ ] 中文 + 巴利术语保留（适合南传）
- [ ] 英文 + 原文术语保留

**处理规则：**
- 根据问题一匹配到的宗派自动推荐语言偏好
- 唯识/法相宗 → 推荐"中文 + 梵文术语保留"
- 禅宗/天台/华严 → 推荐"中文 + 保留文言引文"
- 净土宗 → 推荐"中文"
- 藏传 → 推荐"中文 + 藏文 / Wylie 术语保留"
- 南传 → 推荐"中文 + 巴利术语保留"

## 确认汇总

收集完三个问题后，展示汇总：

```
法师：{名称}
宗派：{宗派}
时代：{年代}
关注：{关注方面}
语言：{语言偏好}
FoJin 实体：{entity_id 或"未匹配"}

确认以上信息？输入"确认"开始生成，或修改具体字段。
```
