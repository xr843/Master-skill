# 三大传统总论

> **何时读这个**：用户问汉传/藏传/南传的差异、宗派定位、某祖师属于哪派、想知道选择哪个传统入门时。
> 本文档供 create-master 主流程在 Step 1 信息录入识别传承时参考，也供回答用户的传统对比问题时调用。

## 三大传统 — 一句话定位

| 传统 | 时空 | 核心特征 |
|------|------|---------|
| **汉传** | 公元 1 世纪起经丝路传入中国，再传朝鲜 / 日本 / 越南 | 大乘为主；汉译三藏 + 本土宗派创立（天台/华严/禅/净/律/唯识/三论/密）；判教传统发达 |
| **藏传** | 7-8 世纪经印度大师入藏（莲花生 / 阿底峡），后弘期定型 | 大乘 + 金刚乘并行；显密双修；活佛转世制度；格鲁/宁玛/萨迦/噶举四大派 |
| **南传** | 公元前 3 世纪阿育王派传至斯里兰卡，后传缅甸/泰国/柬埔寨/老挝 | 上座部独传；以巴利三藏为根本；觉音《清净道论》为论藏统宗；强调出家僧团与内观禅修 |

## 汉传八宗（预置法师对应）

| 宗派 | 核心教义 | 代表法师 | 主依经典 |
|------|---------|---------|---------|
| 三论宗 / 中观 | 缘起性空、中道八不 | 鸠摩罗什 (`/master-kumarajiva`) | 《中论》《百论》《十二门论》《大智度论》 |
| 唯识 / 法相 | 三性三无性、八识、转识成智 | 玄奘 (`/master-xuanzang`) | 《成唯识论》《瑜伽师地论》《摄大乘论》 |
| 天台 | 一念三千、三谛圆融、五时八教、止观双运 | 智顗 (`/master-zhiyi`) | 《法华经》《摩诃止观》《法华玄义》 |
| 华严 | 法界缘起、四法界、十玄六相、一即一切 | 法藏 (`/master-fazang`) | 《华严经》《金师子章》《华严五教章》 |
| 禅宗 | 直指人心、见性成佛、教外别传 | 慧能 (`/master-huineng`)、虚云 (`/master-xuyun`) | 《六祖坛经》《金刚经》《楞伽经》 |
| 净土 | 信愿持名、带业往生 | 印光 (`/master-yinguang`) | 《无量寿经》《观经》《阿弥陀经》 |
| 跨宗派（教宗天台·行归净土） | 性相融会、禅教律净一体 | 蕅益 (`/master-ouyi`) | 《弥陀要解》《教观纲宗》《灵峰宗论》 |

> **律宗 / 真言宗 / 三阶教等暂无预置法师**。如用户问相关宗派祖师（弘一 / 道宣 / 空海 / 信行等），按 create-master 主流程检查 ETHICS Tier 后生成。

## 藏传四派（预置法师对应）

| 派别 | 形成期 | 核心传承 | 代表法师 |
|------|--------|---------|---------|
| 噶当派 | 11 世纪 | 阿底峡《菩提道灯论》、三士道、依止善知识 | 阿底峡 (`/master-atisha`) |
| 噶举派 | 11-12 世纪 | 大手印、那洛六法、口耳传承 | 米拉日巴 (`/master-milarepa`) |
| 萨迦派 | 11 世纪 | 道果教授、显密融贯 | （暂无预置） |
| 格鲁派 | 14-15 世纪 | 宗喀巴《菩提道次第广论》、三主要道、应成中观 | 宗喀巴 (`/master-tsongkhapa`) |
| 宁玛派 | 7-8 世纪奠基，后弘期成形 | 大圆满、伏藏 | （暂无预置） |

> 格鲁派被视为噶当派的复兴形态，故"噶当→格鲁"为道次第传承主线。

## 南传上座部主要禅法传承（预置法师对应）

| 流派 | 起源 | 核心特征 | 代表法师 |
|------|------|---------|---------|
| 大寺派论师传统 (Mahāvihāra) | 5 世纪斯里兰卡 | 《清净道论》七清净十六观智体系 | 觉音 (`/master-buddhaghosa`) |
| 缅甸内观（Mahasi 系） | 20 世纪缅甸 | 标记法、腹部起伏、密集禅修 | 马哈希尊者 (`/master-mahasi-sayadaw`) |
| 泰国森林禅林派 | 20 世纪泰国 | 头陀苦行、自然观察、巴蓬寺传承 | 阿姜查 (`/master-ajahn-chah`) |

> 缅甸系还有列迪 (Ledi Sayadaw) → 乌巴庆 (U Ba Khin) → 葛印卡 (S.N. Goenka) 一脉；泰国森林系还有阿姜曼 (Ajahn Mun) → 阿姜放 (Ajahn Fuang) 等。目前预置覆盖大寺派论藏 + 缅泰两大现代禅修支脉。

## 选择传统的常见判断（用户咨询入门时）

| 用户倾向 | 推荐传统 / 入门法师 |
|---------|-----------------|
| 想读经典原文做学理研究 | 汉传（鸠摩罗什译笈 / 玄奘译笈）+ 南传（觉音《清净道论》） |
| 想要明确禅修指引 | 南传（马哈希标记法 / 阿姜查自然觉知） |
| 想要密法 / 仪轨传承 | 藏传（先三士道入门：阿底峡 / 宗喀巴） |
| 想要顿悟见性的禅风 | 汉传禅宗（慧能 / 虚云） |
| 想要简明易行的解脱道 | 汉传净土（印光） |
| 教理与实修结合 | 汉传天台（智顗）/ 藏传格鲁（宗喀巴） |

> **不评宗派优劣**。以上为基于用户表达需求的引导建议，不构成对传统本身的判断。涉及具体选择，应建议用户亲近善知识。
